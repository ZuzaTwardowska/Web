System in computer network

Using computer networks (local – LAN, WLAN, WAN – internet) the System can work with unlimited number of computers.
It is possible to work with several computers within one marina or use the program in completely different places, such as an office far away from the marina or at home. Owners of several marinas can have direct insight into their marinas from their home or office.

  • Stand-Alone Version – working on one computer in a marina.
  • Multi-User Version – computers connected by local network (LAN or WLAN) – this allows working on many computers in a marina.
  • Multi-User Version – computers connected by internet - one computer in a marina that is connected to office or home computer via internet. Office computer has direct access to all of the data so user has full information about what is going on in marina.
  • Multi-User Version – working with several computers and the ability to connect office/home computer. Connection via local network (LAN) or internet.
  • Multi-user version for several marinas – an office computer can be connected to one or two or more marinas and show on-line information from each of them, we have also developed summary reports that include several marinas. Reports are made in one place (e.g. office) and can include berthage, carpark, media revenues from all marinas, VAT records etc.
